WoltLab Suite Integration (WIP)

Thank you for your interest in the GT Chat extension "WoltLab Suite Integration" by SoftCreatR Media.

Below you will find all the instructions to integrate GT-Chat 0.95+ into your WoltLab Suite based community. To make the installation easier for yourself, you should check the displayed checkbox after each executed step before proceeding to the next step.


Please note:

  1. The instructions below assume both an active (and preferably unmodified) installation of GT-Chat 0.95 or newer, and an active WoltLab Suite installation.
  2. For security reasons, it is necessary that both the chat and WoltLab Suite are accessible under the same domain (.domain.tld).


These instructions are also available in German.
